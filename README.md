# riot-express-ssr

This repo is about SSR of [Riot.js](https://riot.js.org/) with [Express](https://expressjs.com/)

See also
* [@riotjs/ssr](https://github.com/riot/ssr)
* [riot/examples/ssr](https://github.com/riot/examples/blob/gh-pages/ssr/)

## Usage

```bash
# install modules
$ yarn

# boot app
$ yarn start
```
